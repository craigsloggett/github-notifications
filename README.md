# github-notifications
A utility to manage GitHub notifications automatically.

## Environment Variables

| Variable   | Required | Description                                                                                      |
|------------|----------|--------------------------------------------------------------------------------------------------|
| `GH_TOKEN` | Yes      | GitHub personal access token.                                                                    |
| `GH_ORG`   | No       | GitHub org or user to search for open Dependabot PRs that may not appear in your notifications.  |
